请帮我生成审计报告。

## 使用方式

1. **生成完整审计报告**
   ```
   请生成审计报告，任务号：TASK-20240227-001
   格式：Word
   模板：标准审计报告模板
   ```

2. **生成汇总报告**
   ```
   请生成审计汇总报告，包含最近3个月的审计任务
   格式：Excel
   ```

3. **导出整改跟踪报告**
   ```
   请生成整改跟踪报告
   格式：PDF
   ```

## 支持的功能

- 生成完整审计报告（Word/PDF）
- 生成审计汇总统计（Excel）
- 自定义报告模板
- 整改跟踪报告
- 问题分布分析图表
- 历史趋势对比

## 报告类型

| 报告类型 | 说明 | 格式 |
|----------|------|------|
| 详细审计报告 | 包含所有审计项的检查结果 | Word/PDF |
| 执行摘要 | 高层管理摘要 | Word/PDF |
| 问题清单 | 所有发现的问题列表 | Excel |
| 整改跟踪 | 问题整改状态跟踪 | Excel/PDF |
| 统计分析 | 图表和趋势分析 | Excel/PDF |
