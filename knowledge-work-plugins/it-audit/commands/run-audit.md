请帮我执行审计任务并跟踪审计进度。

## 使用方式

1. **创建审计任务**
   ```
   请创建审计任务，包含以下审计项：
   - XC网络安全管理维度的所有审计项
   - XC数据安全管理维度的高风险项
   ```

2. **执行审计任务**
   ```
   请执行审计任务 TASK-20240227-001
   ```

3. **记录审计结果**
   ```
   请记录审计项 ITGC-001 的检查结果：
   - 结果：不符合
   - 发现问题：未启用双因素认证
   - 证据：截图1.png, 配置记录.xlsx
   ```

## 支持的功能

- 创建审计任务（选择审计项、分配人员、设定时间）
- 任务状态管理（待开始/进行中/已完成）
- 审计进度跟踪
- 审计结果记录（符合/不符合/不适用/待确认）
- 证据文件管理
- 问题发现记录与跟踪

## 审计结果类型

| 结果类型 | 说明 |
|----------|------|
| 符合 | 审计项要求已满足 |
| 不符合 | 审计项要求未满足，需整改 |
| 不适用 | 该审计项在当前场景下不适用 |
| 待确认 | 需要进一步核实 |
