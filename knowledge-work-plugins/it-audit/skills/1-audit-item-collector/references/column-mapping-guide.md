# 列名映射参考指南

本文档定义了Excel审计底稿列名到标准字段的映射规则。

## 标准字段定义

### dimension (维度) - 必填

**含义**: 审计项所属的审计领域或主题分类

**常见列名**:
- 一级主题
- 项目
- 审计领域
- 检查类别
- 维度
- 领域
- 主题
- 审计主题
- 检查主题

**示例值**:
- XC信息技术治理
- ITGC-访问控制
- 网络安全管理
- 数据安全

---

### title (标题) - 必填

**含义**: 审计项的具体检查内容描述

**常见列名**:
- 审计项
- 标题
- 检查项
- 问题
- 项目名称
- 检查内容
- 检查要点
- 审计内容

**示例值**:
- 公司是否建立IT治理委员会
- 是否制定网络安全保障方案
- 是否定期进行漏洞扫描

---

### audit_procedure (审计程序)

**含义**: 执行该审计项的具体方法或步骤

**常见列名**:
- 审计程序
- 检查方法
- 检查程序
- 审计方法
- 检查要点
- 审计步骤
- 检查步骤

**示例值**:
- 查阅IT治理委员会成立发文
- 检查公司是否制定建党100年网络安全保障方案
- a)检查公司是否制定自查清单 b)查看检查记录

---

### description (描述)

**含义**: 审计项的详细说明或问题描述

**常见列名**:
- 存在问题
- 检查内容
- 描述
- 问题描述
- 详细描述
- 说明
- 备注

---

### severity (严重程度)

**含义**: 审计项的重要程度或风险等级

**常见列名**:
- 严重程度
- 风险等级
- 重要性
- 优先级
- 风险级别

**标准化映射**:
| 原始值 | 标准值 |
|--------|--------|
| 高、重大、关键、严重、high、critical、major | 高 |
| 中、一般、medium、moderate | 中 |
| 低、轻微、low、minor | 低 |

---

## 不导入的字段

以下字段属于**审计执行结果**，不是审计项定义的一部分，导入时应排除：

| 字段名 | 说明 |
|--------|------|
| 检查结论 | 本次审计的结论(是/否/部分符合) |
| 检查记录 | 审计人员的工作记录 |
| 证据清单 | 收集的审计证据文件列表 |
| 问题 | 发现的具体问题 |
| 整改建议 | 针对问题的改进建议 |
| 责任部门 | 问题责任归属 |
| 整改期限 | 要求整改完成的时间 |

这些字段将在**模块2: 审计执行**中使用。

## 映射配置示例

```python
COLUMN_MAPPING = {
    "dimension": {
        "standard": "dimension",
        "aliases": ["一级主题", "项目", "审计领域", "检查类别", "维度"],
        "required": True
    },
    "title": {
        "standard": "title",
        "aliases": ["审计项", "标题", "检查项", "问题", "项目名称"],
        "required": True
    },
    "audit_procedure": {
        "standard": "audit_procedure",
        "aliases": ["审计程序", "检查方法", "检查程序", "审计方法"],
        "required": False
    },
    "severity": {
        "standard": "severity",
        "aliases": ["严重程度", "风险等级", "重要性"],
        "required": False,
        "default": "中"
    }
}
```

## 常见Excel格式示例

### 格式1: 标准检查底稿

| 一级主题 | 审计项 | 审计程序 | 检查结论 | 证据清单 |
|----------|--------|----------|----------|----------|
| IT治理 | 是否建立IT治理委员会 | 查阅成立发文 | 是 | xxx.pdf |

**映射**: 一级主题→dimension, 审计项→title, 审计程序→audit_procedure

### 格式2: 问题清单

| 检查类别 | 问题描述 | 风险等级 |
|----------|----------|----------|
| 访问控制 | 未定期审查权限 | 高 |

**映射**: 检查类别→dimension, 问题描述→title, 风险等级→severity

### 格式3: 自查表

| 审计领域 | 检查内容 | 检查方法 |
|----------|----------|----------|
| 网络安全 | 是否部署防火墙 | 检查网络拓扑 |

**映射**: 审计领域→dimension, 检查内容→title, 检查方法→audit_procedure
